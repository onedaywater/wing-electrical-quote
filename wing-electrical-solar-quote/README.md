# Wing Electrical Solar Quote

Deploy-ready Next.js project for an online solar quote page.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Import the repo into Vercel.
3. Click deploy.

## Deploy to Netlify

Use the Next.js preset and connect the same repo.

## Notes

- Logo file is already included in `public/logo.png`.
- Company email and GST number are set in `app/page.tsx`.
- Replace the placeholder phone number with your real one before going live.
